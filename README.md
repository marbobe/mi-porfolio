##  Características Principales

* **Diseño Neo-Brutalism:** Estética atrevida con bordes gruesos, sombras duras y colores vibrantes.
* **Internacionalización (i18n):** Soporte completo para Español e Inglés (cambio instantáneo).
* **Totalmente Responsive:** Navbar adaptable con menú hamburguesa y layouts optimizados para móvil.
* **Formulario de Contacto:** Funcional e integrado con **EmailJS** (sin backend propio).
* **UX Mejorada:** Notificaciones Toast personalizadas y avisos de "Cold Start" para servidores en reposo (Render).

---

## Tecnologías Utilizadas

* **Core:** React.js (Vite)
* **Estilos:** Tailwind CSS
* **Iconos:** React Icons (FontAwesome)
* **Traducción:** i18next & react-i18next
* **Formularios:** EmailJS
* **Animaciones:** CSS Transitions & Keyframes personalizadas
